# Neovim Plugin Reference: Complete List of Plugins & GitHub Links

This page lists all the plugins used in the Neovim configuration, powered by [Lazy.nvim](https://github.com/folke/lazy.nvim).

## Package Manager

### Lazy.nvim

- **GitHub**: [folke/lazy.nvim](https://github.com/folke/lazy.nvim)
- **Description**: A modern plugin manager for Neovim with lazy-loading capabilities

## Usage

To install any of these plugins, they should already be configured in your Neovim setup. If you want to add new plugins, you can modify the plugin configuration files in the `lua/plugins` directory.

For more information about plugin configuration, check out the [Basic Setup](/configuration/basic-setup) guide.
