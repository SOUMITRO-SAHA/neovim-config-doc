Tokyo Night is a beautiful and modern Neovim theme that provides a clean, dark color scheme with multiple style variants and extensive customization options.

## What Problem Does It Solve?

Traditional Neovim themes can be:

- Visually inconsistent
- Hard on the eyes
- Limited in customization
- Not modern looking
- Poorly maintained

## How It Improves Productivity

1. **Better Visibility**:

   - Reduced eye strain
   - Clear contrast
   - Consistent colors
   - Better readability

2. **Modern Design**:

   - Clean aesthetics
   - Professional look
   - Multiple styles
   - Beautiful syntax highlighting

3. **Customization**:
   - Multiple variants
   - Color customization
   - Style options
   - Plugin integration

## How to Use

### Basic Setup

- Set colorscheme to "tokyonight"
- Choose style variant
- Configure options
- Enable plugin highlights

### Style Variants

- Night (default)
- Storm
- Day
- Moon
- Custom

### Integration

- Works with many plugins
- Supports terminal colors
- Compatible with airline
- Works with lightline
