# Which-key

[which-key.nvim](https://github.com/folke/which-key.nvim) is a plugin that helps you remember keybindings by showing a popup with possible keybindings of the key you just pressed.

## Key Features

- Keybinding popup
- Group organization
- Custom labels
- Timeout configuration
- Prefix handling
- Register support
- Window customization
- Performance optimized

## Basic Usage

### Default Behavior

- Press a key to see available keybindings
- Wait for timeout to execute command
- Press another key to see nested keybindings
- Press `<Esc>` to cancel

### Navigation

- `<Space>` - Show space keybindings
- `<C-h/j/k/l>` - Show window keybindings
- `z` - Show fold keybindings
- `g` - Show goto keybindings
- `[` - Show previous keybindings
- `]` - Show next keybindings

## Configuration

### Basic Setup

```lua
require('which-key').setup({
  plugins = {
    marks = true,
    registers = true,
    spelling = {
      enabled = true,
      suggestions = 20,
    },
    presets = {
      operators = true,
      motions = true,
      text_objects = true,
      windows = true,
      nav = true,
      z = true,
      g = true,
    },
  },
  icons = {
    breadcrumb = "¬ª",
    separator = "‚ûú",
    group = "+",
  },
  window = {
    border = "single",
    position = "bottom",
    margin = { 1, 0, 1, 0 },
    padding = { 2, 2, 2, 2 },
    winblend = 0,
  },
  layout = {
    height = { min = 4, max = 25 },
    width = { min = 20, max = 50 },
    spacing = 3,
    align = "left",
  },
  ignore_missing = false,
  hidden = { "<silent>", "<cmd>", "<Cmd>", "<CR>", "call", "lua", "^:", "^ "},
  show_help = true,
  triggers = "auto",
  triggers_blacklist = {
    i = { "j", "k" },
    v = { "j", "k" },
  },
})
```

### Custom Groups

Define custom keybinding groups:

```lua
local wk = require('which-key')

wk.register({
  f = {
    name = "file",
    f = { "<cmd>Telescope find_files<cr>", "Find File" },
    r = { "<cmd>Telescope oldfiles<cr>", "Open Recent File" },
    n = { "<cmd>enew<cr>", "New File" },
  },
  b = {
    name = "buffer",
    b = { "<cmd>Telescope buffers<cr>", "Show Buffers" },
    n = { "<cmd>bnext<cr>", "Next Buffer" },
    p = { "<cmd>bprevious<cr>", "Previous Buffer" },
    d = { "<cmd>bdelete<cr>", "Delete Buffer" },
  },
  g = {
    name = "git",
    s = { "<cmd>Git<cr>", "Status" },
    c = { "<cmd>Git commit<cr>", "Commit" },
    p = { "<cmd>Git push<cr>", "Push" },
    l = { "<cmd>Git pull<cr>", "Pull" },
  },
}, { prefix = "<leader>" })
```

### Window Customization

Customize the popup window:

```lua
require('which-key').setup({
  window = {
    border = "rounded",
    position = "bottom",
    margin = { 1, 0, 1, 0 },
    padding = { 2, 2, 2, 2 },
    winblend = 0,
    zindex = 1000,
  },
  layout = {
    height = { min = 4, max = 25 },
    width = { min = 20, max = 50 },
    spacing = 3,
    align = "left",
  },
})
```

## Advanced Features

Add custom labels to keybindings:

```lua

  prefix = "<leader>",
  icons = {
    breadcrumb = "¬ª",
    separator = "‚ûú",

```

### Timeout Configuration

Configure popup timing:

```lua
require('which-key').setup({
  timeout = 300,
  delay = 0,
  triggers = "auto",
  triggers_blacklist = {
    i = { "j", "k" },
    v = { "j", "k" },
  },
})
```

### Register Support

Enable register display:

```lua
require('which-key').setup({
  plugins = {
    registers = true,
  },
  register = {
    bindings = {
      ["'"] = "quotes",
      ['"'] = "double quotes",
      ['`'] = "backticks",
      ['('] = "parenthesis",
      ['['] = "square brackets",
      ['<'] = "angle brackets",
      ['{'] = "braces",
    },
  },
})
```
